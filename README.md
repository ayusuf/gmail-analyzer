# Gmail Analyzer

Created this program to analyze my gmail.
It connects via imap.
Originally you could use just your gmail password but that has been [disabled](https://support.google.com/accounts/answer/6010255?hl=en) so you can create something called an app password if you don't use the API.

## How to use

The way I'm using it right now is through IntelliJ and passing my email and app password CLI args.
